# XFCatBot
## 介绍
适用于Xenforo的Python机器人。原本给自己朋友论坛用的，然后因为太多人想要传过来的。
## 部署
1. 安装Python3.6+
2. 安装依赖：
```bash
pip install requests
pip install json
pip install toml
pip install datetime
```
3. 根据配置文件的内容，修改配置文件。
4. 打开plugin_example.py，修改插件代码实现您的功能。
5. 重命名plugin_example.py为plugin.py，config_example.toml为config.toml。
6. 运行start.py。
至此，你的机器人已成功运行。
